## 六、字体授权协议详解

### 6.1 SIL Open Font License 1.1（SIL OFL 1.1）

最常见的开源字体协议，绝大多数开源字体采用。

| ✅ 允许 | ❌ 禁止 |
|----------|----------|
| 免费使用（含所有商业用途） | 单独销售字体文件本身 |
| 嵌入软件、App、网页、电子书 | 使用原名字发布修改版（Reserved Font Name） |
| 修改字体 | |
| 再分发（含修改版，须用 OFL 协议） | |
| 印刷、广告、Logo 中使用 | |
| 随商业产品打包分发 | |

**关键条款**：
- **Reserved Font Name**：衍生字体不能使用原字体名称，需改名发布
- **Copyleft 性质**：修改后的字体也必须在 OFL 下发布
- **不能单独卖字体文件**：但可以随产品/服务一起打包

### 6.2 Apache License 2.0

| ✅ 允许 | 说明 |
|----------|------|
| 免费商用 | 含闭源使用 |
| 修改和再分发 | 无 Copyleft 限制 |
| 可再授权（Sublicense） | 不同于 OFL |
| 无 Reserved Font Name | 更灵活 |

### 6.3 MIT License

| ✅ 允许 | 说明 |
|----------|------|
| 完全自由使用 | 含商用 |
| 修改和闭源分发 | 最宽松 |
| 无任何名称限制 | |

### 6.4 IPA Font License v1.0

- 日本 IPA（独立行政法人情報処理推進機構）发布
- OSI 批准的开放源代码协议
- 允许商用、修改、嵌入
- 衍生字体必须使用同一协议，且不得使用"IPA"在名称中

### 6.5 Ubuntu Font License

- Ubuntu 字体系列专用协议
- 允许免费使用和再分发
- 修改版需改名，文档中需包含声明

### 6.6 GPL（含字体嵌入例外）

- 文泉驿等字体使用
- 可商用，但需注意嵌入式场景下的协议约束
- "Font Exception" 条款允许在不要求文档也 GPL 的情况下嵌入字体

### 6.7 CC 协议

- **CC0**：完全放弃版权，任意使用/修改/商用，无需署名
- **CC BY 4.0**：可商用可修改，需署名原作者
- **CC BY-SA 4.0**：可商用可修改，需署名 + 修改版需同样 CC BY-SA

### 6.8 品牌自有免费商用协议

阿里巴巴、华为、小米、OPPO 等品牌自行拟定的免费商用授权：
- 通常允许：印刷、嵌入、网页、App 内使用
- 通常限制：不可单独转售字体文件、不可盗用品牌名义
- 使用时建议保存授权页面截图

## 八、使用注意事项

1. **优先从官方渠道下载** —— 确保文件完整和授权清晰，避免第三方二次打包篡改或植入病毒
2. **核对字体包内 LICENSE 文件** —— 官网标注为 OFL，但字体包内可能另有详细条款，以包内授权文件为准
3. **保存授权截图/文件至项目档案** —— 商业项目必备，防止版权纠纷时无法举证
4. **注意 Reserved Font Name 条款** —— 修改字体后必须改名发布，不能使用原名字
5. **禁止单独转售字体文件** —— OFL 允许随产品分发（如打包在软件安装包中），但严禁把 TTF/OTF 文件单独出售
6. **"免费下载" ≠ "免费商用"** —— 务必确认授权协议后再用于商业项目
7. **平台限制 ≠ 全领域免费** —— 汉仪部分字体仅限京东/阿里平台使用，并非全领域免费商用
8. **方正 GB2312 版免费 ≠ GBK 版免费** —— 只下载 GB2312 字符集版本
9. **系统自带字体可能有版权风险** —— Windows 的黑体/宋体等不要提取出来商用
10. **GPL 字体的"传染性"** —— 如果在软件中嵌入 GPL 字体，整个软件的授权可能受 GPL 约束
11. **商标注册需额外注意** —— 部分字体允许一般商用但不允许用于商标注册（如 斗鱼追光体、胡晓波系列）
12. **微信公众号关注授权** —— 部分字体（三极、润植家）需要关注公众号获取正式授权证书
13. **OFL 字体最安全** —— 优先选择 SIL OFL 1.1 或 Apache 2.0 协议的字体，法律风险最低
